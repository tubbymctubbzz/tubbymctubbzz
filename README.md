## 🚀 **Featured Project**

<div align="center">

### 🛠️ **Radnt CLI**
**Next.js Project Generator**

A powerful CLI tool for creating Next.js projects with shadcn/ui components and modern development tools. Get up and running with a beautiful, production-ready application in seconds.

**🔥 Key Features:**
- 🏗️ Multiple project templates (Basic, Dashboard, E-commerce, Blog, Portfolio)
- 🎨 Pre-configured shadcn/ui components
- ⚡ TypeScript & Tailwind CSS setup
- 🔧 ESLint & Prettier configured
- 🚀 Production-ready deployment setup
- 📱 Responsive design out of the box

### 🚀 **Quick Start**

```bash
# Install globally
npm install -g radnt-cli

# Create a new project
radnt create my-app

# Start development
cd my-app && radnt dev
```

**Or use with npx:**
```bash
npx radnt-cli create my-app
cd my-app && npm run dev
```
